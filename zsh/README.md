# Simple ZSH configuration


## Installation

Install Oh My Zsh.

```sh
sh -c "$(curl -fsSL https://raw.githubusercontent.com/robbyrussell/oh-my-zsh/master/tools/install.sh)"
```

Copy `.oh-my-zsh.custom`, `.aliases` and `.zshrc` to your home directory.

```sh
cp -r \
  .oh-my-zsh.custom \
  .aliases \
  .zshrc \
  ~
```
